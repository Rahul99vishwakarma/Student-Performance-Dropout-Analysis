# 🎓 Student Performance & Academic Risk Analysis

> **An End-to-End Data Analytics Project using Python, SQL, SQLite, Plotly & Machine Learning**

---

## 📖 Project Overview

This project focuses on analyzing student academic performance to discover the factors that influence final grades. The complete data analytics pipeline includes data cleaning, exploratory data analysis (EDA), SQL analysis, interactive visualizations, machine learning, and logging.

The objective is to transform raw educational data into meaningful insights that can help identify students who may require additional academic support.

---

## 🎯 Project Objectives

- Analyze student academic performance
- Clean and preprocess real-world data
- Perform Exploratory Data Analysis (EDA)
- Execute SQL queries using SQLite
- Build interactive dashboards with Plotly
- Develop a Machine Learning model for performance prediction
- Generate actionable insights for decision-making

## 💡 Key Insights

- Students with higher study time generally achieve better grades.
- Previous academic failures negatively affect final performance.
- G1 and G2 are the strongest predictors of G3.
- Attendance plays a role in student success.
- Most students belong to GP School.
